# 🛍️ ShopSphere - E-Commerce Platform

ShopSphere is a modern e-commerce web application built using React, TypeScript, Tailwind CSS, Supabase, and Vercel.

## 🚀 Live Demo

https://e-commerce-six-flax-46.vercel.app

## 📂 GitHub Repository

https://github.com/sivatejaswini932/e-commerce

---

## ✨ Features

- Product Listing
- Category Browsing
- Product Search
- Featured Products
- Trending Products
- Responsive Design
- Supabase Database Integration
- Real-Time Data Fetching
- Vercel Deployment

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sivatejaswini932/e-commerce.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build project:

```bash
npm run build
```
